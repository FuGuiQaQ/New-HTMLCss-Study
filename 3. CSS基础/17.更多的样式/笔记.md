# 更多的样式

## 透明度

1. opacity, 设置的是整个元素的透明度 它的取值是0~1
2. 在颜色位置设置alpha通道(rgba)

## 鼠标

使用cursor设置

## 盒子隐藏

1. display:none 不生成盒子
2. visibility:hidden 生成盒子,只是从视觉上移除盒子

## 背景图

## img跟背景图的区别

img元素是属于HTML概念

背景图属于css的概念

1. 当图片属于网页内容时,必须使用img元素
2. 当图片仅用于网页的美化时,必须使用背景图
